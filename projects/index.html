Modern Web Application (JavaScript, Python, Flask, AWS)
I built a dynamic and flexible web app using a bunch of different AWS services. The app was hosted on a front-end web server using Fargate and Docker and connected to a backend DynamoDB database. I also added a user registration and authentication feature to analyze user interactions and behavior. I was inspired to tackle this project to learn to use some tools commonly used in industry.

Cube Renderer (Java, Python)
As part of a hackathon, I worked with my team to create a Java program to render a rotating cube. We created a matrix quaternion module to perform math calculations and a custom cube mesh to generate the skeleton of the cube. We also wrote a port of the program in Python. This was one of the coolest projects I've worked on since I was able to create a really cool graphical thing.
